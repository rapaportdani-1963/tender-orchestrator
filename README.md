# 🤖 Tender Orchestrator

מערכת ניהול מכרזים אוטומטית — קוראת נתונים מ-Google Sheets ומפעילה סוכני AI דרך Claude API.

## 🚀 התקנה מהירה

### שלב 1 — העלאה ל-GitHub Pages

1. צור Repository חדש ב-GitHub (למשל `tender-orchestrator`)
2. העלה את שני הקבצים:
   - `index.html`
   - `manifest.json`
3. היכנס ל-**Settings** → **Pages**
4. תחת **Source** בחר: `Deploy from a branch`
5. תחת **Branch** בחר: `main` ותיקייה `/ (root)`
6. לחץ **Save**
7. תוך דקה האפליקציה תהיה זמינה בכתובת:
   `https://YOUR_USERNAME.github.io/tender-orchestrator/`

### שלב 2 — הגדרת המערכת

1. פתח את האפליקציה בדפדפן
2. לחץ על **⚙️ הגדרות**
3. הזן:
   - **שם ואימייל** — פרטי המשתמש
   - **Google Sheets** — הדבק את הקישור המלא של כל גיליון
   - **Claude API Key** — מפתח מ-[console.anthropic.com](https://console.anthropic.com/settings/keys)
4. לחץ **💾 שמור הגדרות**

### שלב 3 — הכנת Google Sheets

כל גיליון צריך להיות משותף לצפייה:

1. פתח את הגיליון
2. לחץ **שיתוף** (Share)
3. שנה ל-**"כל מי שיש לו קישור"** (Anyone with the link)
4. ודא שההרשאה היא **צופה** (Viewer)
